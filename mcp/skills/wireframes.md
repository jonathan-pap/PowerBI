# Skill: Wireframes вҖ” Report Layout Patterns

All layouts use canvas **1280 Г— 720** (16:9), `displayOption: "FitToPage"`.
Standard margin: **10px** on all sides. Gap between elements: **10px**.
Z-order: create background shapes first (lower z), data visuals on top (higher z).

## Z-order Convention

- **1000** вҖ” background panels, header bar
- **2000вҖ“5000** вҖ” content area shapes
- **5000вҖ“10000** вҖ” foreground data visuals
- **11000+** вҖ” overlays, labels, badges

Always create shapes **before** data visuals so z-order stays correct automatically.
